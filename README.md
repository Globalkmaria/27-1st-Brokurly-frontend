# Brokurly

!! 컬리만의 유기농 채소로 집에서도 간편하게 즐겨보컬리?! !! 라는 비젼을 가지고 컬리로 시작하여 컬리로 끝날 수 있는 서비스를 제공하고 있습니다.

## 구현 기능

- 회원가입
- 로그인
- 제품 리스트
- 제품 상세
- 장바구니
- 구매 내역
